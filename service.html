Zadaci

  • 1. Množenje brojeva

    Korisnik unosi dva broja i program ispisuje proizvod ta dva broja.

  • 2. Formula

    Korisnik unosi dva broja x i y i ispisuje (x+y)^2

  • 3. Djeljenje

    Korisnik unosi dva decimalna broja x i y. Program ispisuje a/b.

  • 4. Eksponencijalna funkcija

    Korisnik unosi broj x, program ispisuje koliko je 2^x

  • 5. Veći broj

    Korisnik unosi dva broja, program ispisuje koji je od ta dva broja veći.

  • 6. Paran broj

    Korisnik unosi broj, program ispisuje dali je broj paran

  • 7. Hipotenuza na kvadrat

    Korisnik unosi katete a i b pravouglog trougla i ispisuje vrijednost njegove hipotenuze c na kvadrat (ispisati c^2)

  • 8. Trougao *

    Korisnik unosi tri stranice a, b i c i ispisuje dali te tri stranice mogu ciniti trougao.

  • 9. Dolari u kovanicama*

    Korisnik unosi decimalni broj koji predstavlja iznos u dolarima, program ga konvertuje u cente i ispisuje najmanji broj kovanicama potreban za taj iznos. (NPR. ako unese 2.81 to je 281 centi i to su 5 kovanica po 50centi, jedna kovanica od 25centi, jedna kovanica od 5centi i jedna kovanica od 1cent, program ce ispisati broj 8 jer je 5+1+1+1=8).

  • 10. Najveći broj*

    Korisnik unosi brojeve sve dok ne unese nulu. Nakon što unese nulu ispiše se najveći broj što je korisnik unijeo.

  • 11. Zbir brojeva*

    Korisnik unosi brojeve sve dok ne unese nulu. Nakon što unese nulu ispiše zbir svih brojeva što je korisnik unijeo.

  • 12. Prost broj*

    Korisnik unosi broj, program ispisuje dali je broj prost ili nije.

  • 13. Slučajan broj napredni *

    Korisnik unosi dva broja, program ispisuje slučajno odabrani broj od manje do većeg broja što je korisnik unijeo.

  • 14. Ocjena

    Korisnik unosi broj od 1 do 5. Program ispisuje ocjenu na osnovu broja (1 - nedovoljan, 2 - dovoljan, 3 - dobar, 4- vrlo dobar, 5 - odličan). Ako korisnik nije unijeo broj od 1 do 5 ispisuje se poruka "Unos nije tačan".

  • 15. Površina pravouglog trougla

    Korisnik unosi dužine kateta pravouglog trougla i ispisuje kolika mu je površina.

  • 16. Zbir sa uslovom

    Korisnik unosi tri broja, program ispisuje dali je zbir prva dva broja veći od trećeg.

  • 17. Veći broj sa dodatnim uslovom

    Korisnik unosi dva broja, program ispisuje dali je prvi broj dva puta veći od drugog broja.

  • 18. Ispisivanje broja

    Korisnik unosi broj i program ga ispisuje 20 puta zaredom (NPR. ako unese 5 ispisuje se 55555555555555555555)

  • 19. Formula 2

    Korisnik unosi broj a, ispisuje koliko je 10*a-12+a*a*a

  • 20. Slučajan broj

    Program ispisuje slučajan broj od 1 do 6.
