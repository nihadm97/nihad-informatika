Zadaci

  • 1. Lista

    Korisnik unosi n brojeva, nakon sto unese sve brojeve ispisuju se svi uneseni brojevi od prvog do zadnjeg

  • 2. Lista od zadnjeg

    Korisnik unosi n brojeva, nakon sto unese sve brojeve ispisuju se svi uneseni brojevi od zadnjeg do prvog

  • 3. Lista parni

    Korisnik unosi n brojeva, nakon sto unese sve brojeve ispisuju se svi parni brojevi od prvog do zadnjeg

  • 4. Lista prosti *

    Korisnik unosi n brojeva, nakon sto unese sve brojeve ispisuju se svi prosti brojevi od prvog do zadnjeg

  • 5. Lista prosti 2*

    Korisnik unosi n brojeva, nakon sto unese sve brojeve ispisuju se zadnji prosti broj koji je unesen

  • 6. Trostruki

    Korisnik unosi n brojeva a1, a2, ... , an, nakon sto unese sve brojeve ispisuju se svi uneseni brojevi pomnozeni sa tri

  • 7. Zbir

    Korisnik unosi broj n, nakon cega se unosi n brojeva, program ispisuje zbir tih n brojeva

  • 8. Aritmetička sredina

    Korisnik unosi broj n, nakon cega se unosi n brojeva, program ispisuje aritmetičku sredinu unesenih brojeva

  • 9. Dolari u kovanicama*

    Korisnik unosi decimalni broj koji predstavlja iznos u dolarima, program ga konvertuje u cente i ispisuje najmanji broj kovanicama potreban za taj iznos. (NPR. ako unese 2.81 to je 281 centi i to su 5 kovanica po 50centi, jedna kovanica od 25centi, jedna kovanica od 5centi i jedna kovanica od 1cent, program ce ispisati broj 8 jer je 5+1+1+1=8).

  • 10. Najveći broj*

    Korisnik unosi brojeve sve dok ne unese nulu. Nakon što unese nulu ispiše se najveći broj što je korisnik unijeo.

  • 11. Zbir brojeva*

    Korisnik unosi brojeve sve dok ne unese nulu. Nakon što unese nulu ispiše zbir svih brojeva što je korisnik unijeo.

  • 12. Prost broj*

    Korisnik unosi broj, program ispisuje dali je broj prost ili nije.

  • 13. Slučajan broj napredni *

    Korisnik unosi dva broja n i m, program ispisuje tacno n slucajnih brojeva od m/2 do m ako je m paran odnosno n slucajnih brojeva od (m-1)/2 do m

  • 14. Ocjena*

    Korisnik unosi broj n, nakon cega unosi n brojeva od 1 do 5. Program ispisuje zakljucnu ocjenu na osnovu n unesenih ocjena

  • 15. Dan u sedmici ***

    Korisnik unosi datum, program ispisuje koji je to dan u sedmici

  • 16. Zbir sa uslovom

    Korisnik unosi broj djeljiv sa tri, program ispisuje dali je zbir prve 2/3 brojeva veci od zbira ostatka brojeva.

  • 17. Najmanji paran broj

    Korisnik unosi broj n, nakon toga se unosi n brojeva, program ispisuje najmanji paran broj od unesenih n brojeva.

  • 18. Zbir + razlika*

    Korisnik unosi brojeve m i n (m < n), nakon cega se unosi n brojeva, ispisuje se zbir prvih m brojeva minus zbir ostalih brojeva

  • 19. Formula 2*

    Korisnik unosi brojeve ai, ispisuje koliko je a1-a2+a3-a4+...-an

  • 20. Slučajan broj

    Program ispisuje n slucajnih brojeva od 1 do 6.
